# 環境構築

## 1. リポジトリのクローン

```bash
git clone https://github.com/yoshiyuki-140/fast-api-lesson.git
cd fast-api-lesson # ディレクトリ移動
```

## 2. Dockerの起動

```bash
docker-compose up -d
```
